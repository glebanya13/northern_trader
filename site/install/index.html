Установка APK на Android

  1. Скачайте файл по кнопке выше.
  2. Откройте уведомление о загрузке или найдите файл в «Загрузки» / «Файлы».
  3. Нажмите на .apk и подтвердите установку.
  4. Если система блокирует установку: «Настройки» → «Безопасность» / «Приложения» → разрешите установку для браузера или для этого источника (формулировка зависит от версии Android).

Ярлык на рабочий стол (Android)

После установки приложение обычно появляется в списке приложений. Чтобы вынести иконку на главный экран:

  • Стандартный лаунчер: долгое нажатие на иконку приложения в списке приложений → перетащите на свободное место на главном экране.
  • Через виджеты/меню: на пустом месте главного экрана долгое нажатие → «Виджеты» / «Приложения» (название зависит от производителя) → выберите приложение.

Если вы открываете сайт в Chrome, а не нативное приложение: меню браузера (⋮) → «Установить приложение» / «Добавить на главный экран» — появится ярлык на веб-приложение (если сайт поддерживает PWA).

iPhone (ярлык на сайт)

Установка APK на iOS невозможна. Если нужен быстрый доступ к сайту: Safari → кнопка «Поделиться» → «На экран «Домой»».
